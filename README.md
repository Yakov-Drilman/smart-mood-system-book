## 🧭 מה זה הפרויקט?

Smart Mood System הוא פרויקט שמחבר בין ספר טכנולוגי לבין בניית מערכת תוכנה מלאה.

הוא מתעד תהליך מלא של בניית מערכת שמתרגמת רעיונות ורגשות למערכת דיגיטלית חכמה.
